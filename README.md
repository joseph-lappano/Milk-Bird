# Milk Bird
A Python/pygame remake of Flappy Bird

Features:
- Game loop with physics and scoring
- 20 unique levels
- Volume Controls
- Level progress reset

Controls:
- Space: Jump/Flap
- G: Glide
- Mouse/Escape: Menu Controls

# Requirements
- Python 3.12+
- pygame 2.6.1+

# How to Run
'''bash
pip install -r requirements.txt  
python src/main.py
